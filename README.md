# PortSwigger Web Security Academy Labs

## Overview

This repository contains my hands-on practice, notes, and lab documentation from the PortSwigger Web Security Academy. The goal of this repository is to document practical web application security testing techniques, vulnerability analysis, exploitation methodologies, and remediation strategies while improving my understanding of offensive and defensive security concepts.

The repository is structured according to different learning paths and vulnerability categories covered during the labs.

---

# Objectives

- Develop practical web application penetration testing skills
- Improve understanding of common web vulnerabilities
- Practice HTTP request manipulation and response analysis
- Learn secure coding and remediation techniques
- Build a documented cybersecurity learning portfolio
